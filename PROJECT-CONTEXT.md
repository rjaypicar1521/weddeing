## What We Are Building

**Wedding-Online** is a Filipino-first online wedding invitation platform.

Couples build a cinematic, animated digital wedding invitation. Guests access it via a secret code, RSVP, and leave wishes. The invitation becomes a permanent memory archive.

**Unique angle:** Invitation → Live Wedding Day Hub → Permanent Memory Archive

**Target market:** Filipino couples (culturally-aware: Ninong/Ninang, GCash QR, regional templates)

## Authentication Architecture

### Couple Auth (Sanctum SPA)
```
1. GET /sanctum/csrf-cookie         → Sets XSRF-TOKEN cookie
2. POST /auth/login                 → Sets laravel_session cookie
3. All requests: withCredentials:true, X-XSRF-TOKEN header (Axios auto-handles)
```

### Guest Auth (Stateless JWT)
```
1. POST /guest/validate-code { code: "X7K2P9" }
   ← { guest_token: "eyJ..." }  (24hr expiry)
2. Store in sessionStorage
3. All guest requests: Authorization: Bearer {guest_token}
```

### Admin Auth
```
Same as Couple Sanctum auth
+ AdminOnly middleware checks users.is_admin = true
```

## Key Business Rules

1. **One invitation per user** — `users` has one `invitation` (1:1)
2. **Guest code is 6-char alphanumeric** — uppercase, no ambiguous chars (0,O,1,I,L) — e.g., `X7K2P9`
3. **Slug ≠ Guest code** — `/i/[slug]` is the permanent URL; guest code is entered on that page
4. **One RSVP per guest JWT** — enforced in `RSVPController` (check before insert)
5. **Invitations never expire** — `status` can be `archived` but URL always resolves
6. **Storage quotas**: Free = 50MB, Premium = 500MB (enforced by `CheckStorageQuota` middleware)
7. **All uploads converted to WebP** before R2 upload (except QR codes → PNG)
8. **Music = YouTube URL only** (MVP) — no file upload for audio
9. **Prenup video = YouTube URL only** — no file upload for video
10. **Maps = Google Embed API** (free iframe) + Waze deeplink (no JS Maps SDK)

## Guest Invitation Page — Section Order

The `/i/[slug]/view` page renders in this exact scroll order:

1. Cinematic intro overlay (3–4s, skippable)
2. Prenup video (YouTube embed, full-width)
3. Wedding details (names, date, time, venue — hero typography)
4. Countdown timer (flip animation)
5. Love story timeline (vertical, chapter cards)
6. Wedding day schedule/program
7. Venue section (Google Maps Embed + Waze + Google Maps deeplinks)
8. Entourage gallery (grouped by role)
9. Dress code (color swatches + text)
10. Cash gift / QR codes section
11. RSVP form (multi-step, 4 steps)
12. Boarding pass confirmation (shown after RSVP submit)
13. Wishes wall (masonry quote cards)
